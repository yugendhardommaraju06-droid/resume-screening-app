# AI Resume Screening System

AI Resume Screening System is a Flask-based web application that analyzes resumes and matches them with job descriptions using AI-powered screening.

## Features
- Resume Upload
- Job Description Matching
- Candidate Skill Analysis
- Match Score Generation
- Clean Web Interface
